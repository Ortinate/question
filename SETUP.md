# Supabase Setup Instructions

To enable the professional backend for this questionnaire:

1.  **Create a Supabase Project**:
    *   Go to [supabase.com](https://supabase.com) and sign up/log in.
    *   Create a new project.

2.  **Create the Database Table**:
    *   Go to the "SQL Editor" in your Supabase dashboard.
    *   Run the following SQL query:
        ```sql
        create table questionnaire_responses (
          id bigint generated by default as identity primary key,
          created_at timestamp with time zone default timezone('utc'::text, now()) not null,
          data jsonb not null
        );
        ```

3.  **Get Credentials**:
    *   Go to **Project Settings** -> **API**.
    *   Copy the `Project URL` and `anon` `public` key.

4.  **Connect the App**:
    *   In your project folder, create a file named `.env.local` (copy from `.env.local.example`).
    *   Add your credentials:
        ```
        VITE_SUPABASE_URL=your_project_url
        VITE_SUPABASE_ANON_KEY=your_anon_key
        ```
    *   Restart the development server (`npm run dev`).

The admin dashboard will now fetch data directly from your real database!
